Dictionary of Command Line Cheats...For Cheaters
COMMAND FUNCTION
cal: pronounced, 'KaL' after you type in the three middle letters of the word scalp...the terminal will display a calendar
clear: pronounced, 'kuh-leer' will utterly erradicate anything in the terminal window
cd: pronounced, 'kuh-d' allows really smart and attractive developers to change the directory they are working in
CNTRL + C: pronounced, 'kontrl see' allows you to exit the shell, though I don't know why you'd want to as you should always be working
cp pronounced, 'kippp' creates a copy of a file and tucks it away safely into a specified destination
mkdir: pronounced, 'mick-durr' this micks durrs, meaning it makes directories
rmdir: pronounced, 'rimm-durrr' this rims durrs, as in it removes directories
rm -rf: pronounced 'rimmm-ruff' despite it having 'ruff' in its name it has little to do with the noise a dog makes, instead it will delete the contents of the entire directory
pwd: pronounced 'pyuood' it sounds goofy when you say it outloud, but cut it some slack because it prints your working directory
diff: shows difference between 2 files
mv: pronounced 'muvvvvvv' it will move a file to wherever you demand it should be moved!
ls: pronounced 'lisssss', will list the files within a directory
